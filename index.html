HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element is important for SEO
  • SEO stand for Search Engine Optimization and helps improve the ranking of a webpage in search engine sites.
  • The section element defines thematic content.
  • src defines location of an image and alt describes the image.
  • alt can contribute to SEO
  • The footer element contains information on the author, copyright, contact, sitemap, and navigation of a webpage.

CSS

  • A maragin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS to style directly in HTML; good for quick and permant changes. Must be edited individually.
  • Internal Style Sheet are rules for the head section of HTML. Used to style multiple elements in a page.
  • External CSS Style Sheet is most common way to style. Allows for all CSS rules in a seperate file.
  • A selector defines the element or attributes to which the rules or declerations apply.
  • Use * to indicate all elements apply.
  • A link is an element that can connect to external files.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A branch-name: creates modifications to branch and adds all changes
  • git pull origin branch-name: to see modifications and pulls to Github repo
  • git push: to push changes to Github

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • After declaring a variable give it a value.
  • Types of variables: String, Number, Boolean, Array
  • Comments are text added to code as notes/explinations.
  • Opperators are a mathmatical symbol that creates results based on two values or variables.
  • Conditionals are code structures used to test if an expression returns true or not.
  • Undefines indicates the abscence of a value.
  • null indicates the abscence or an object.
  • Loops are used to perform repeated tasks based on a condition.
  • An Array is a group of related data. A variable must be declared to use, and not all arrays must have multiple data types.
  • An array index is the location of data in an array by unique number. Always start with 0.
  • A function is a set of instructions that tells the computer how to perform a certain task. Must be called by name to execute.
